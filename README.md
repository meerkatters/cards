# cards

cards and such.

## card_counter.py

A game to practice card counting (lo-hi method)

### Usage

`$ chmod 770 card_counter.py`
`$ python card_counter.py -h`

User will be promted at terminal:

- enter to deal a new card
- c to view the count
- q to exit
